## Getting Started

- Clone this repository by running `https://github.com/TalhaAkhun/Portfolio-Template-2.git` in your CLI.
- Type `cd Portfolio-Template-2`
- Type `code .`
- Run `npm install`
- Run `npm start`
- Open Browser `http://localhost:3000/`
